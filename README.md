# Проект высоконагруженного сервиса Сбербанк Онлайн
## 1. Тема и целевая аудитория
### Описание и аналоги
#### Описание [[1]](https://www.sberbank.ru/ru/person/dist_services)
Сбербанк Онлайн — удаленный канал обслуживания Банка, автоматизированная защищенная система дистанционного обслуживания Клиента через Официальный сайт Банка в сети Интернет, а также Мобильное приложение Банка. Сбербанк Онлайн позволяет получать подробную информацию о банковских услугах: счетах, вкладах, банковских картах и кредитах, а также погашать кредиты, совершать платежи, переводы между вкладами и счетами банковских карт, переводить деньги клиентам Сбербанка и других банков. Помимо этого, в канал обсуживания интегрировано большое количество иных сервисов Сбербанка, включая сервисы для инвестирования, получения документов, контроля и оплаты услуг ЖКХ, использования развлекательных сервисов экосистемы Сбера и пр.
Сервис работает как в виде интернет-кабинета через браузер, так и в виде мобильного приложения для Android и iPhone.

#### Аналоги [[2]](https://www.rustore.ru/prostore/banki-obzor-prilozhenii-1)
У сервиса есть ряд аналогов – как зарубежных, так и отечественных. Перечислены сервисы, которые также интегрировали большое количество услуг в свои приложения, помимо тех, которые напрямую относятся к управлению финансами. Основные конкуренты в настоящий момент — отечественные (работа россиян с зарубежными банками осложнена): ТБанк, ВТБ Онлайн, Альфа Банк, Озон Банк, Почта Банк, Точка Банк и др.

### Объём хранилища
+ Среднее количество транзакций в минуту:
  `34 722 + 1 526 = 36 248`
+ Средний размер одного чека равен 60 кБ. Тогда за минуту генерируется:
  `36 248 * 60 кБ = 2 174 880 кБ = 2,074 ГБ` информации, которая заполняет память
+ Объём данных, генерируемых за день транзакциями (сделаем поправку 0,74 на ночное время в часовом поясе МСК) [[12]](https://ru.wikipedia.org/wiki/Население_России):
  `2,074 ГБ * 60 * 24 * 0,74 = 2 210,0544 ГБ`
+ Ежемесячный объём данных по транзакциям:
  `2 210,0544 ГБ * 30 = 66 301,632 ГБ = 64,75 ТБ`
+ Годовой объём данных по транзакциями:
  `64,75 ТБ * 12 = 777 ТБ`
+ Также данные пользователей хранятся в профилях весом 512 кБ:
  `700 кБ * 84 300 000 = 659,48 ТБ`
+ Всего:
  `777 ТБ + 659,49 ТБ = 1436,49 ТБ = 1,4 ПБ`
